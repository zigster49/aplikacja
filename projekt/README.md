


## Jak uruchomić

```bash
npm install
npm start
# lub podczas developmentu:
npm run dev
```

Otwórz przeglądarkę: **http://localhost:3000**

## Endpointy API

| Metoda | Ścieżka        | Opis                        |
|--------|----------------|-----------------------------|
| POST   | /api/login     | Logowanie użytkownika       |
| POST   | /api/register  | Rejestracja użytkownika     |
